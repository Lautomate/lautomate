Workforce Perusal

Level: Standard

Tools: Excel, Power BI, and SQL

Topic: Workforce

I developed a workforce project in Power BI, starting with data loaded from Excel into SQL. I created a view for employees to simplify access and improve efficiency, which serves as a dimension table in my data modeling. I also established a date dimension table by writing code in a blank query before loading the data, to enhance time-based analysis. I imported the data directly targeting the view that I previously created in SQL.

To provide additional context, I created a calculated table for age dimensions, categorizing the data into meaningful age brackets. Utilizing DAX, I implemented basic yet effective techniques, such as tooltips for age ranges, to enhance user experience. Additionally, I incorporated a dimension table to track the number of target employees per position, utilizing the Enter Data feature in Power BI for straightforward data entry.
