YouTube Library

Level: Intermediate

Tools and Techniques: Python, Power BI, Power Query, Data Modeling, and DAX

Topic: YouTube Channels

NOTE

This project replicates an actual business scenario but does not utilize the company's real data. The YouTube channel referenced in this project serves as a sample and should be replaced with the channels relevant to your company, such as product-related channels.

GOAL

The goal of this project is to create a centralized view that integrates data from multiple YouTube channels, allowing users to analyze and monitor channel performance without the need to visit each channel individually. By consolidating this information into a single Power BI report, the project streamlines the process of data analysis, saving time and enabling more efficient decision-making based on real-time insights.

INTRODUCTION

I developed a comprehensive YouTube project in Power BI, leveraging Python to scrape data from multiple YouTube channels. The Python scripts are directly integrated with Power BI, allowing for seamless data integration and real-time updates. I utilized key libraries and frameworks for web scraping and asynchronous data retrieval, ensuring that the data was collected efficiently and quickly.

PROCESS

I began by identifying all the YouTube feed URLs for the channels included in this project. Using Power Query, I gathered all the YouTube feed URLs and performed necessary transformations. This step was straightforward, as the feed is structured. However, the YouTube feed provides incomplete data, necessitating additional methods to obtain other important information.

To address this, I employed Python and its libraries to expedite the process. I imported and installed the required libraries and frameworks, compiling a list of all the YouTube channel URLs. By combining asyncio and aiohttp, I enabled concurrent requests to all the specified URLs, allowing the project to process multiple requests simultaneously rather than sequentially.

After receiving the data, I utilized Beautiful Soup to parse the HTML documents according to the criteria defined in my code. I selected specific tags and elements to retain valuable data while discarding irrelevant information. This library facilitated the web scraping process, making it efficient and manageable.

However, a significant challenge when working with various YouTube channels without using an API is the inconsistency in data structure. To overcome this, I employed Python's Regular Expressions to match patterns, manipulate, and extract specific text accurately. Although this step was complex, it proved rewarding once I successfully identified the patterns across different channels.

One of my objectives was to run the script directly in Power BI rather than on my local desktop. To achieve this, I put all my code in Power BI using the Python Script connector, utilizing the Pandas library to structure all my datasets into DataFrames, as required by Power BI. I also included Matplotlib, even though it was not used in this project, as it is a dependency for Power BI integration.

After extracting data from various sources, I performed few transformations in Power Query, including merging datasets, replacing values, and creating conditional columns.

LIMITATION

A maximum of 15 videos can be retrieved per channel.

RESULT

This project successfully integrated data from multiple YouTube channels into Power BI, providing a comprehensive view of channel performance. The Python scripts efficiently scraped the necessary data in less than 30 seconds, overcoming challenges related to inconsistent data structures across different channels.

The final Power BI report included key performance indicators (KPIs) such as total views, subscriber growth, video engagement, and content performance across all sampled YouTube channels. The data visualizations provided clear and actionable insights, enabling users to understand trends and patterns in the data effectively.

Despite the limitation of retrieving a maximum of 15 videos per channel, the report provided a strong analysis of channel performance, demonstrating the value of automated data collection and integration processes in a business context.
