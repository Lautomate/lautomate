YouTube Library

Level: Intermediate

Tools: Python, Power BI, Power Query, Data Modeling, DAX

Topic: YouTube

I developed a comprehensive YouTube project report in Power BI, leveraging Python to scrape data from multiple YouTube channels. The Python scripts are directly connected to Power BI, allowing for seamless data integration and real-time updates. I utilized important libraries and frameworks related to web scraping and asynchronous retrieval, ensuring the data was collected as efficiently and quickly as possible.

Additionally, I used Power Query to connect to YouTube channel data, automating the transformation and cleaning processes while retaining important columns needed for data exploration and insights.
